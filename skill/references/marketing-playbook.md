# Marketing Playbook

## The Core Story

This game's biggest marketing asset is its origin story: **a team of AI agents
collaboratively built a production-quality game through democratic consensus.**

This story is inherently viral because it triggers multiple emotional responses:
curiosity ("can AI actually do this?"), amazement ("wow, this is good"), and
debate ("what does this mean for game development?").

---

## Pre-Launch Campaign (Weeks 1-8)

### Week 1-2: Seed the Story
- Create a public development dashboard showing real-time agent activity
- First Twitter/X thread: "We're letting AI agents collaboratively build a game.
  Here's what happened in the first 48 hours..."
- Set up Discord with channels: #agent-decisions, #gameplay-previews, #vote-with-us
- Reddit posts in r/gamedev, r/artificial, r/indiegaming

### Week 3-4: Build Anticipation
- Weekly development updates (with screenshots and agent decision highlights)
- "Agent Debates" content series — show the agents disagreeing about design choices
- Let Discord community vote on key design decisions alongside agents
- Start a dev blog documenting the technical architecture

### Week 5-6: Show, Don't Tell
- First gameplay footage (short clips optimized for social media)
- Behind-the-scenes technical deep dives
- Guest posts on game dev and AI publications
- Podcast appearances on indie game and AI shows

### Week 7-8: Pre-Launch Hype
- Beta access waitlist
- Influencer preview copies (game dev YouTubers, AI creators)
- Press kit distribution
- Landing page with email capture
- Countdown to launch

## Launch Campaign

### Launch Day
- Coordinated social media push across all platforms
- Press embargo lifts — reviews and coverage go live
- Discord community event
- Reddit AMA: "We built a game with AI agents — AMA"
- Product Hunt launch
- Hacker News submission

### Launch Week
- Respond to all press and social media mentions
- Daily highlight reels of player moments
- Community challenges and competitions
- Bug fix rapid response (agents + human team)

## Growth Mechanics (Built Into the Game)

### Viral Loops
1. **Share Replay** — one-tap share of your best moments as short video clips
2. **Challenge a Friend** — send a challenge with your score to beat
3. **Creation Sharing** — if the game has UGC, easy sharing of creations
4. **Leaderboard Screenshots** — auto-generated shareable leaderboard cards
5. **Achievement Cards** — beautiful shareable graphics for milestones

### Social Features
1. **Weekly Challenges** — global challenges that create shared conversation
2. **Seasonal Events** — time-limited content that drives FOMO and return visits
3. **Clan/Guild System** — social groups for retention
4. **Spectator Mode** — watch skilled players (content creation enabler)

### Content Creator Support
1. **Replay System** — full match replay for YouTube/Twitch content
2. **Custom Thumbnails** — auto-generated thumbnail-worthy screenshots
3. **API Access** — let creators build tools and trackers
4. **Creator Program** — early access to new features for content creators

## Platform Distribution Strategy

### Tier 1 (Launch)
- **Web** (direct URL) — zero friction, instant play
- **itch.io** — indie game community, discovery potential
- **PWA** — installable on mobile from web

### Tier 2 (Month 1-2)
- **Newgrounds** — large gaming audience, supports web games
- **CrazyGames / Poki** — web game portals with large audiences
- **Social media embeds** — playable in-feed where supported

### Tier 3 (Month 3+, if metrics justify)
- **Steam** — requires quality threshold, provides huge audience
- **Mobile app stores** — native wrapper around PWA or full native port
- **Epic Games Store** — if the AI story angle gets traction

## Metrics to Track

### Acquisition
- Website visitors → signup conversion rate
- Social media impressions → click-through rate
- Press coverage reach
- Organic search traffic
- Referral traffic (viral coefficient)

### Activation
- First session completion rate
- Tutorial completion rate
- Time to "aha moment" (first meaningful achievement)
- First social share

### Retention
- D1 / D7 / D30 retention
- Session frequency and length
- Feature adoption rates
- Churn analysis (why do players leave?)

### Revenue (if applicable)
- Conversion to paid (if freemium)
- ARPU / ARPPU
- LTV / CAC ratio

### Virality
- K-factor (invites sent × conversion rate)
- Social shares per session
- Organic mentions and word-of-mouth
- Content creator coverage

## Budget Considerations

This project can launch with near-zero budget:
- Hosting: Cloudflare Pages free tier
- Domain: ~$12/year
- Analytics: PostHog free tier or self-hosted
- CI/CD: GitHub Actions free tier
- Community: Discord free
- Marketing: Organic + the AI story as hook

If budget becomes available, prioritize:
1. Server infrastructure for multiplayer
2. Content creator partnerships
3. Paid social media promotion
4. Conference/event presence
5. Localization services
